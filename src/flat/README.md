# flat
展开函数

### 使用方法

```javascript
  // 如果环境支持则使用 `Array.prototype.flat`
  // array为被展开数组， depth为展开层级，默认为 1
  flat(array, depth = 1)
  
  flat([1,2,3]) // 结果： [1,2,3]
  flat([1,2,3,[4,5,6]]) // 结果： [1,2,3,4,5,6]
  flat([1,2,3,[4,5,6,[7,8,9]]]) // 结果： [1,2,3,4,5,6,[7,8,9]]
  flat([1,2,3,[4,5,6,[7,8,9]]], 2) // 结果： [1,2,3,4,5,6,7,8,9]
```
